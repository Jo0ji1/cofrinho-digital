Assets required for the Cofrinho Digital app:

- icon.png (1024x1024) - App icon
- splash.png - Splash screen image
- adaptive-icon.png - Android adaptive icon foreground
- notification-icon.png - Android notification icon

These image files must be added before building the app.
Recommended: Use a piggy bank / wallet icon with the primary color #10B981 (emerald green).
